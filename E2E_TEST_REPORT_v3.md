## API 端点汇总

### tRPC 路由结构
- `crf.subject.list` - 获取受试者列表
- `crf.subject.getById` - 获取单个受试者
- `crf.subject.create` - 创建受试者
- `crf.subject.update` - 更新受试者
- `crf.subject.delete` - 删除受试者（软删除）
- `crf.intervention.list` / `upsert` - 干预记录
- `crf.weight.list` / `upsert` - 体重记录
- `crf.gripStrength.list` / `upsert` - 握力记录
- `crf.pgsga.list` / `upsert` - PG-SGA评估
- `crf.mfsi.list` / `upsert` - MFSI疲劳量表
- `crf.appetite.list` / `upsert` - 食欲记录
- `crf.inflammation.list` / `upsert` - 炎症指标

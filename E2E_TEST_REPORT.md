## 已知问题

### 1. 后端枚举限制 ⚠️
- `tumorType` 只能是: 口咽癌、食管癌、胃癌、肝癌、胆囊癌、胰腺癌、结直肠癌、其他
- **注意**: 前端下拉选项中不应包含"肺癌"等不在枚举中的选项

- 后端 API 返回的中文字段（如 studyName）存在编码问题
- 影响: 显示乱码，但不影响数据存储
